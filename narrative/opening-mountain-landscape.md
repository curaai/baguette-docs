# 절 주변 씬 배치

## 구성 요소

- 주변 산과 숲
- 절로 이어지는 산길
- 절 입구
- 중앙 마당
- 대웅전
- 요사채
- 명부전
- 건물과 마당을 잇는 돌길
- 높이가 다른 마당과 건물을 연결하는 돌계단
- 요사채 뒤쪽의 화장실과 창고

## 배치

```text
                  가파른 산비탈

             [ 명부전 ]   [ 대웅전 ]
                    돌계단

         [ 요사채 ] ── [ 중앙 마당 ]
              │                │
      [ 화장실·창고 ]      [ 절 입구 ]
                               │
                             산길
```

## 주변 구성

- 절 뒤쪽은 가파른 숲으로 막아 이동 범위를 제한한다.
- 절 양옆은 나무와 비탈로 자연스러운 경계를 만든다.
- 절 앞쪽은 산길과 입구만 열어 진입 방향을 명확하게 보여준다.
- 대웅전과 명부전은 마당보다 높은 곳에 두고 돌계단으로 연결한다.
- 요사채는 중앙 마당 옆의 같은 높이에 둔다.
- 화장실과 창고는 요사채 뒤쪽의 눈에 덜 띄는 공간에 둔다.
